# Obsidian
## Atajos de teclado
v 0.1
### Acciones comunes

| Acción            | Atajo                             |
|-------------------|-----------------------------------|
| Copiar            | `Ctrl+C`                          |
| Cortar            | `Ctrl+X`                          |
| Pegar             | `Ctrl+V`                          |
| Pegar sin formato | `Ctrl+Shift+V`                    |
| Deshacer          | `Ctrl+Z`                          |
| Rehacer           | `Ctrl+Shift+Z` or `Ctrl+Y`        |
| Copiar párrafo    | `Ctrl+C` (sin texto seleccionado) |
| Cortar parafo     | `Ctrl+X` (sin texto seleccionado) |

### Edición de texto

| Acción                        | Atajo                                   |
|-------------------------------|-----------------------------------------|
| Insertar nueva línea          | `Enter`                                 |
| Elimina el carácter anterior  | `Retroceso`                             |
| Elimina el carácter siguiente | `Supr`                                  |
| Elimina palabra anterior      | `Ctrl+Retroceso`                        |
| Elimina palabra siguiente     | `Ctrl+Supr`                             |
| Elimina la línea actual       | `Ctrl+Shift+K` (sin texto seleccionado) |

### Movimiento del cursor 

| Acción                                              | Atajo                   |
|-----------------------------------------------------|-------------------------|
| Mueve el cursor un carácter                         | `Flecha Izq/Der`        |
| Mueve el cursor al principio de la palabra anterior | `Ctrl+Flecha izquierda` |
| Mueve el cursor al final de la palabra siguiente    | `Ctrl+Flecha derecha`   |
| Mueve el cursor al principio de la línea actual.    | `Inicio`                |
| Mueve el cursor al final de la línea actual.        | `Fin`                   |
| Mueve el cursor a la línea anterior                 | `Flecha arriba`         |
| Mueve el cursor la línea siguiente                  | `Flecha abajo`          |
| Mueve el cursor al principio de la nota.            | `Ctrl+Inicio`           |
| Mueve el cursor al final de la nota.                | `Ctrl+Fin`              |
| Mueve el cursor una página hacia arriba.            | `AvPág`                 |
| Mueve el cursor una página abajo.                   | `RePág`                 |

### Selección de texto

| Acción                                                          | Atajo                   |
|-----------------------------------------------------------------|-------------------------|
| Quitar selección                                                | `Escape`                |
| Seleccionar todo                                                | `Ctrl+A`                |
| Extiende la selección un carácter                               | `Shift+Flecha Izq/der`  |
| Extiende la selección hasta el principio de la palabra anterior | `Ctrl+Shift+Flecha Izq` |
| Extiende la selección hasta el final de la siguiente palabra    | `Ctrl+Shift+Flecha Der` |
| Extender la selección hasta el principio de la línea actual     | `Shift+Inicio`          |
| Extender la selección hasta el final de la línea actual         | `Shift+Fin`             |
| Extienda la selección hasta el principio de la nota             | `Ctrl+Shift+Inicio`     |
| Extienda la selección hasta el final de la nota.                | `Ctrl+Shift+Fin`        |
| Amplíe la selección una página hacia arriba.                    | `Shift+AvPág`           |
| Amplíe la selección hacia abajo                                 | `Shift+RePág`           |
