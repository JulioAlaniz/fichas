# Fichas y guías de aplicaciones

Para dar soporte a usuarios de aplicaicones del ámbito del **software libre**. Por intermedio de Malal Libros Editorial desplegamos esta iniciativa: fichas de atajos de teclado, guías de comandos y tips de uso.
Como usuario sentí (y siento muchas veces) la necesidad de un instrumento como este. Si bien existen infinidad de sitios y publicaciones, a la hora de buscarlas producen demoras que retrasan nuestro flujo de trabajo.
